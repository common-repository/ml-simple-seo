With ML Simple SEO You can edit title tag and add description tag to any post or page.

== Description ==

ML Simple SEO is plugin which allows add own title and description tag to any post or page which is use in blog or website. Thats is very usefull when Googlebot indexing your blog or website. Look to "Screenshots" part or to home page of plugin ([click!](http://www.mlipinski.pl/plugins/ml-simple-seo/)) to get more info.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to edit post or page in admin menu. Below a WYSWIG editor appeared two new input areas. Here you can edit title and description tags.
